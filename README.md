プロジェクトタイトル
===========================
多数のJUNOS機器(MX/EX/SRX)に同じ設定を流しこんだり、設定内容を確認するための  
PyEZを使ったツールのサンプルです。

## Requirement(前提)    
* Linux(Ubuntu)  
* Webサーバー(Apache2)  
* FTPサーバー  
* PyEZ

## How to install(インストール方法)  
1. Linux + Webサーバ + FTPサーバを準備する
2. PyEZをインストールする
3. ファイルをコピーする
4. プログラムの初期化を実行
5. ブラウザを立ち上げて、プログラムにアクセスする
